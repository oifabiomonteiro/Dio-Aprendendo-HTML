Sobre a tag P

voltar

A tag "p" é utilizada para demarcar um parágrafo. O elemento HTML < p > (ou Elemento HTML Parágrafo) representa um parágrafo do texto. Parágrafos são, geralmente, representados em mídia visual , como bloco de texto que são separados dos blocos adjacente por espaços brancos verticais e/ou recuo de primeira-linha.

Resumindo, eles servem para da uma organizada nos seus texto.

Sobre as tags H

voltar

São recursos de programação HTML utilizados para destacar títulos e subtítulos de uma página. Elas são utilizadas na hierarquização e estrutura de uma página. Elas facilitam a leitura, entendimento e até mesmo a indexação do conteúdo e sua utilização é bem mais simples do que parece.

Basicamente elas servem para aumentar ou diminuir um texto, palavras ou uma letra.

Exemplo:

  • H1

  • H2

  • H3

  • H4

  • H5
  • H6

Sobre a tag OL

voltar

Listas ordenadas. Uma lista ordenada é também uma lista de itens. As listas de itens são marcadas com números. Uma lista ordenada começa com a tag (ol).

Basicamente elas servem para ordenar de forma numerica os itens.

Exemplo:

  1. primeiro
  2. segundo
  3. terceiro
  4. quarto
  5. quinto
  6. sexto

Sobre a tag UL

voltar

A tag (ul)é um elemento HTML que representa uma lista não ordenada. Este elemento tem como característica criar listas não ordenadas no HTML, ou seja, ao invés de apresentar uma lista com numeração a lista será apresentada com marcadores.

Basicamente elas servem para ordenar com marcadores os itens.

Exemplo:

  • primeiro
  • segundo
  • terceiro
  • quarto
  • quinto
  • sexto
